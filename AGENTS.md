# Project: Gift-Forms (B2B Influencer Gifting)
## Tech Stack
- React, Tailwind CSS, TypeScript.

## Rules for Codex
- **Vibe:** Minimalist, premium, B2B-focused.
- **Workflow:** Create small, modular components. Do not overwrite large files.
- **Context:** We are building an influencer gifting platform. Influencers need mobile-first UI.
- **Naming:** Use camelCase for variables and PascalCase for components.
